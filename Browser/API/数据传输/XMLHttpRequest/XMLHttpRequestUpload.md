# XMLHttpRequestUpload

`XMLHttpRequestUpload` 接口代表特定 `XMLHttpRequest` 的上传过程。它是一个不透明的对象，表示底层依赖于浏览器的上传过程。该接口继承自 `XMLHttpRequestEventTarget`，可通过调用 `XMLHttpRequest.upload` 来获取。
